A Python-based cybersecurity tool that analyzes URLs and identifies potential security risks.

## Features

* HTTPS verification
* Suspicious keyword detection
* Domain extension analysis
* URL length analysis
* Risk score generation
* Low / Medium / High risk classification

## Technologies Used

* Python
* Lists
* Loops
* Conditional Statements
