# Password Generator

## Overview

This is a simple Python program that generates random passwords based on the length entered by the user.

## Features

* Generates random passwords
* User chooses password length
* Uses letters and numbers

## Technologies Used

* Python
* random module

## What I Learned

* Taking user input
* Using loops
* Generating random values in Python

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

python password_generator.py

## Example

Enter password length: 8

Generated Password: A7k2Pq9X

## Purpose

This project demonstrates a basic cybersecurity concept: strong and random passwords are more secure than simple or predictable passwords.
