## About

A Python-based cybersecurity tool that generates secure passwords and evaluates their strength based on password length.

## Features

* Custom password length
* Optional numbers
* Optional special characters
* Password strength assessment (Weak, Medium, Strong)

## Technologies Used

* Python
* Loops
* Conditional Statements
* Random Module
