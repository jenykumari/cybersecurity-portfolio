## About

A Python-based cybersecurity tool that analyzes IPv4 addresses and classifies them as:

* Loopback IP Address
* Private IP Address
* Public IP Address
* Unspecified Address

It also validates IP format and provides a basic security assessment.

## Features

* IP address validation
* Risk level classification
* Detection of loopback, private, public, and unspecified addresses
* Beginner-friendly cybersecurity project

## Technologies Used

* Python
* Conditional Statements
* Loops
* String Manipulation
